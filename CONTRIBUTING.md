# 贡献指南

感谢你愿意参与故影 Memory Portrait。

这个项目既包含工程问题,也包含记忆、隐私和伦理问题。贡献时请优先尊重真实人物和家庭感受。

## 可以贡献什么

- 改进 README 和文档。
- 设计更好的记忆采访问卷。
- 完善人物档案 schema。
- 实现无 GPU 的前端 MVP。
- 实现生成器 provider。
- 增加测试和示例。
- 改进伦理、安全和隐私说明。

## 提交原则

- 示例数据必须是虚构的。
- 不提交未经授权的真实人物照片或家庭资料。
- 不加入人脸搜索、身份识别或监控能力。
- 不把 AI 生成图描述为真实历史照片。
- 涉及隐私、导出和分享的改动应在 PR 中明确说明风险。

## 建议 PR 拆分

优先提交小而清楚的 PR:

1. 文档或 schema 改进。
2. 前端静态页面。
3. Mock API。
4. 会话状态和导出逻辑。
5. 生成器 provider。

项目当前处于设计阶段。代码结构确定后,本文档会补充具体启动命令。
