# Roadmap

## Phase 0: 文档与设计

- 明确项目愿景、非目标和伦理边界。
- 定义人物记忆档案 schema。
- 准备虚构示例档案。
- 建立贡献指南。

## Phase 1: 无 GPU MVP

- Web 表单: 创建人物记忆档案。
- 候选图网格: 使用 Mock 数据展示 6-9 张候选。
- 选择反馈: 支持更像、不像、备注、返回上一代。
- 会话 API: 创建、选择、重随机、完成。
- 导出: YAML + Markdown + 图片占位。

## Phase 2: StyleGAN Provider

- 接入 StyleGAN2/3 预训练模型。
- 实现初始隐空间采样。
- 实现选择、交叉、变异。
- 评估年龄、发型、脸型等属性方向稳定性。
- 加入基础生成质量评测。

## Phase 3: 体验增强

- 多人协作记忆。
- 局部特征锁定实验。
- 版本对比和历史回退。
- 公开分享前隐私检查。

## Phase 4: 高清重绘与档案发布

- 可选扩散模型 refine。
- 生成带水印和元数据的最终图。
- 支持静态纪念页面导出。
- 完善多语言文档。
