# Ethics Reference

## Non-Negotiables

- Do not present outputs as restored photographs.
- Do not identify, locate, match, or search for real people.
- Do not generate evidence for law enforcement, journalism, litigation, immigration, or official records.
- Do not help impersonate a person or create deceptive historical material.
- Do not publicly share sensitive family or community details unless the user confirms appropriate consent.

## Safer Framing

Use:

```text
memory-based memorial portrait
AI-generated remembrance image
one possible visual interpretation of shared memories
```

Avoid:

```text
real photo
accurate reconstruction
true face
proof
identification
```

## Consent Checklist

Before public sharing, ask the user to confirm:

- The person is deceased or the user has consent for a living person.
- The image will be labeled as AI-generated.
- Sensitive personal details have been removed.
- Family or community norms allow this kind of memorial image.
- The image will not be used for identity claims or evidence.

## Handling Risky Requests

If the user asks for a disallowed use, refuse that use briefly and redirect:

```text
I cannot help create an image for identification or evidence. I can help create a clearly labeled memory portrait archive for private remembrance instead.
```

## Minors and Schools

For remembered classmates or school settings:

- Avoid sexualized, humiliating, or sensational presentation.
- Prefer adult-age remembered versions when possible.
- Keep names and school details private unless explicitly authorized.

## Cultural Sensitivity

Respect mourning customs. If the user mentions religious or cultural constraints around images of the deceased, prioritize text archives, symbolic imagery, or non-facial memorial artifacts.
